Troubleshooting

Hiddify not connecting

Start with the simple checks first. Most Hiddify connection problems are caused by mode, subscription status or a stale configuration.

1. Check whether the subscription is active

If your subscription expired, Hiddify may still show the imported profile, but the connection will not work. Open @simplelinkaccessbot, go to My Configs and check the expiration date.

If it expired, press Extend. You normally do not need to import the link again after renewal.

2. Refresh the subscription

After renewal or provider-side updates, press Refresh in Hiddify. This asks Hiddify to download the latest version of your subscription link. A refresh is faster and cleaner than deleting the profile and importing it again.

3. Use VPN mode

If you are not sure which connection mode is selected, switch to VPN mode. Proxy and System Proxy modes can be valid, but some applications ignore them. VPN mode is the best test because it routes traffic more broadly.

4. Restart Hiddify after a timeout

On a first connection attempt, Hiddify can occasionally take longer while establishing the connection. Press Connect again. If it remains on a timeout screen, close Hiddify completely, open it and connect again.

5. Test your base internet connection

Disconnect Hiddify and check whether normal websites open. If the base connection is slow, unstable or captive-portal blocked, Hiddify cannot fix that network layer by itself.

6. Check per-app routing

On Android, Per-app Proxy can route some apps through Hiddify and others directly. If only one app fails, check whether it is included in the Hiddify route. Temporarily disable Per-app Proxy to test.

Fastest recovery path: renew if needed, refresh the subscription, switch to VPN mode, restart Hiddify and connect again.
