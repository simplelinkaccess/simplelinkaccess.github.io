What is Hiddify?

If you are new to modern internet access technologies, Hiddify is the best place to start.

Hiddify is a free, open-source application that lets you connect to technologies like VLESS with a QR code or a subscription link. There is no complicated setup, no server management and no technical knowledge required.

Why we recommend it

Free and open source

Hiddify is developed as an open-source project. There are no client subscriptions, vendor lock-in or hidden restrictions. You own your configuration and can use it with any compatible provider.

Available on every device

The same client is available for Android, iPhone, iPad, Windows, macOS and Linux. Learn it once, then use it everywhere.

Built for modern protocols

Hiddify supports VLESS, Reality, TUIC, Hysteria 2, WireGuard and many other modern protocols out of the box. You do not need a different application for every protocol.

Designed for everyday use

  • Ad blocking
  • IPv6 support
  • Split tunneling
  • Multiple profiles
  • QR code import
  • Automatic updates
  • One-tap connect
Why do we not build our own VPN app? Because Hiddify is already an excellent client. Instead of reinventing it, we focus on helping you master it and making modern internet access simple for everyone.
