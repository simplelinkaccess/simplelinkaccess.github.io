What is a VLESS subscription?

A VLESS subscription is a link that gives a compatible client, such as Hiddify, the connection configurations it needs.

Many people first meet VLESS through a QR code or a long subscription URL. That can look technical, but the idea is simple: Hiddify is the application on your device, and the VLESS subscription is the configuration source Hiddify reads from.

The subscription can include server addresses, protocol settings, security parameters, display names and routing information. You normally do not edit those details by hand. You import the subscription once, then the client uses it to create a profile you can connect with.

Client, protocol and subscription

It helps to separate three concepts. The client is the app you install. Hiddify is a client. The protocol is the technology used for the connection. VLESS is one of those protocols. The subscription is the delivery format that lets the client receive one or more usable configurations.

This separation is useful because your subscription is not tied to one proprietary app. If a client supports VLESS subscriptions, it can usually use the same link.

Why subscriptions are easier than manual setup

Manual configuration asks users to copy many separate values: address, port, UUID, security mode, transport, hostnames and other protocol details. A subscription link packages that information so the client can import it automatically.

That also makes updates easier. If a provider changes servers or adjusts configuration, you can refresh the subscription in Hiddify instead of deleting everything and starting again.

How SimpleLinkAccess uses subscriptions

SimpleLinkAccess creates your VLESS subscription automatically after payment with Telegram Stars. You receive the subscription in the bot and can import it with a QR code or by copying the link.

Once imported, Hiddify stores the profile locally. Renewing the subscription does not usually require importing it again. You keep the same link and refresh it if Hiddify needs to download the latest configuration.

Short version: Hiddify is the app. VLESS is the connection technology. Your subscription link tells Hiddify how to connect.
