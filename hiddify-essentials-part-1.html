Hiddify Essentials, Part 1

Before importing your VLESS subscription, spend a few minutes configuring Hiddify.

Most people install the app, add a profile and press Connect. That works, but Hiddify can do much more. The most important setting to understand first is connection mode.

Connection mode

Connection mode controls how traffic from your device goes through Hiddify. You can find it on the Home screen, below the main connection button.

Proxy

Only apps that explicitly support proxy settings will use Hiddify. This mode is useful for advanced users, browsers, Telegram or apps where you manually configure a proxy. Most apps will ignore it.

System Proxy

Hiddify sets a proxy for the operating system. Many desktop apps and browsers will use it automatically, but some apps may still ignore system proxy settings.

VPN

Hiddify creates a VPN tunnel and routes device traffic through it. This is the easiest and most reliable mode for most users.

Recommended modes

  • Android: VPN
  • iPhone and iPad: VPN
  • Windows: VPN or System Proxy
  • macOS: VPN or System Proxy
  • Linux: VPN or System Proxy
If you are not sure what to choose, start with VPN. On desktop systems, System Proxy is also a comfortable fallback.
